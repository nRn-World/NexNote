# Security Policy

## Supported Versions

Currently, the following versions of NexNote are supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you find a security vulnerability, please do NOT open a public issue. Instead, please report it by sending an email to bynrnworld@gmail.com.

We will acknowledge your report within 48 hours and keep you updated on the progress of a fix.
